# Linux Server Health Monitor
A Bash script that generates a server health report.
## Features
- Displays hostname
- Displays current data and time
- Displays disk usage
- Displays memory usage
- Displays Nginx service status
## Commands Used
- hostname
- date
- df -h
- free -h
- systemctl status nginx
##  Skills Demonstrated
- Linux command line
- Bash scripting
- System monitoriing
- Service management
- Output redirection
